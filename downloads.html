What's included

Once you download and extract the Glacier zip file, you'll get:


├── Glacier.pdf
├── LICENSE
├── README.md
├── SHA256SUMS
├── SHA256SUMS.sig
├── base58.py
└── glacierscript.py

This includes the Glacier scripts and PDF file with the Glacier protocol. It also includes checksum and signature files for you to validate if the package you got can be trusted. Learn how to use these files.
